# 게임 컨셉
- ### High Concept
    점프를 통해 장애물을 뛰어넘어 목적지에 도달하여 클리어 하는 기록형 게임 (Geometry Dash 모작)

- ### 핵심 메카닉
    기본적으로 플레이어 캐릭터는 맵 좌->우 로 이동한다. \
    화면의 아무 곳이나 터치하면 플레이어 캐릭터가 일정 높이 점프하여 지형지물을 극복하고 장애물을 회피한다. \
    플레이어-지형지물, 플레이어-장애물, 플레이어-상호작용 패달의 충돌 체크.
